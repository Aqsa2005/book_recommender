### Project Description
A simple web app built with Python and Flask that recommends books based on mood and genre. It shows all matching books and highlights one random recommendation.

### The Problem Statement
Readers often struggle to find books that match their mood or genre preference.

### The Solution
This app filters books by mood and genre, displays all matches, and highlights one random book to make discovery easier.

- **View All:** Displays the entire book list.
